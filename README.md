# AI & ML Based Stock Prediction System

## Overview
The AI & ML Based Stock Prediction System is a Flask-based web application that predicts future stock prices using Machine Learning. It analyzes historical stock market data, provides real-time stock information, and displays graphical visualizations to help users understand market trends.

## Features
- User Registration and Login
- Real-Time Stock Data
- Stock Price Prediction
- Interactive Graphs
- Download Stock Data
- Responsive User Interface

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- HTML
- CSS
- Bootstrap
- JavaScript
- Yahoo Finance API

## Installation

1. Clone the repository
```bash
git clone https://github.com/BanashankariNagaraja/AI-ML-Based-Stock-Prediction-System.git
```

2. Install the required packages
```bash
pip install -r requirements.txt
```

3. Run the application
```bash
python app.py
```
